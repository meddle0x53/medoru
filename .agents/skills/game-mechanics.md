## ELO Rating System
- K-factor: 32
- Expected score: `1 / (1 + 10^((opponent_rating - player_rating) / 400))`
- New rating: `rating + K * (actual_score - expected_score)`
- Actual score: 1 for win, 0.5 for draw, 0 for loss

## Real-time Sync
- Use `Phoenix.PubSub` topic: `"duel:#{duel_id}"`
- Broadcast events: `:question_revealed`, `:answer_submitted`, `:time_update`, `:duel_ended`
- Client handles optimistic UI updates, server validates
