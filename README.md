# Medoru

A social Japanese learning platform built with Phoenix LiveView.

## Features

- **Kanji Learning** - Browse and study kanji with animated stroke order diagrams
- **Vocabulary** - Learn words with proper readings and meanings
- **Lessons** - Structured lessons for progressive learning
- **Daily Reviews** - SRS-based review system to maintain your knowledge
- **Stroke Animation** - Visual stroke order with interactive playback
- **User Profiles** - Track your progress and achievements

## Getting Started

To start your Phoenix server:

* Run `mix setup` to install and setup dependencies
* Start Phoenix endpoint with `mix phx.server` or inside IEx with `iex -S mix phx.server`

Now you can visit [`localhost:4000`](http://localhost:4000) from your browser.
