# v7 Lesson Design - Topic-Based Progressive Lessons

## Overview
300 lessons total: 100 N5, 100 N4, 100 N3
Each lesson: 15-20 words (avg 16) = ~4800 words planned

## Lesson Type Distribution
- `:vocabulary` - Core lessons with word learning (70%)
- `:reading` - Reading comprehension with example sentences (15%)
- `:grammar` - Grammar patterns using learned words (15%)

## Lesson Structure

### Vocabulary Lesson (15-20 words)
```json
{
  "title": "Basic Greetings",
  "description": "Essential Japanese greetings for daily conversation",
  "difficulty": 5,
  "order": 1,
  "word_count": 15,
  "word_ids": ["..."],
  "prerequisite_lesson_id": null,
  "lesson_type": "vocabulary"
}
```

### Reading Lesson
- Uses example sentences from words
- 3-5 short reading passages
- Questions based on content

### Grammar Lesson
- 3-5 grammar patterns
- Uses previously learned vocabulary
- Example sentences with explanations

## Progression Rules
1. Lessons unlock sequentially within a level
2. Must complete 80% of N5 before unlocking N4
3. Each lesson unlocks after previous is completed
4. Vocabulary lessons unlock reading/grammar lessons
